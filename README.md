# Credit Card Check Application 

by Russell Adcock

http://au.linkedin.com/in/russelladcock

Requires Ruby 1.9.3

Command Line Version
--------------------

Run the following commands.

- bundle install
- rake db:migrate
- rake test

Web Version
-----------

Run the following commands.

- rake db:seed
- rake cucumber
- rails server
- then navigate to http://localhost:3000/
